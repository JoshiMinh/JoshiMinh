  # Flappy Bird
</div>

## Dependencies

- Latest [Java](https://www.java.com/download/ie_manual.jsp)
- Java Development Kit ([JDK](https://www.oracle.com/java/technologies/downloads/#jdk21-windows))

## How to Play

1.  **Compile the game**:
    -   Run `build.bat`. This will compile the source code and place the `.class` files in the `bin` directory.
2.  **Run the game**:
    -   Run `run.bat`. This will start the game.
3.  **Choose your options**:
    -   **Player Name**: Enter your name. It will be saved for the next time you play.
    -   **Themes**: Choose your preferred theme.
    -   **Difficulty**:
        -   Easy
        -   Normal
        -   Hard
        -   Impossible (not recommended)
4.  Click **ScoreBoard** to view the online database scoreboard.
5.  Click **PLAY** or **EXIT**.
