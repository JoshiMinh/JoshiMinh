Project Overview

This project explores passenger survival on the Titanic using feature engineering and model comparison (Logistic Regression, KNN, SVM, Naive Bayes, Decision Tree, Random Forest). It consolidates detailed passenger data and creates useful features like FarePerPerson, Age_Class, and compact labels for AgeGroup, FareGroup, and Deck.

See the full analysis, visuals, feature engineering, and model selection in the notebook.

Key Engineered Features

  • FarePerPerson = Fare ÷ FamilySize
  • Age_Class = Age × Pclass
  • Deck (A–G, T, U) unified from cabin indicators
  • Encoded: Sex, Embarked, Title, FareGroup, AgeGroup

Notes

  • The deck consolidation maps cabin indicators to a single Deck label (A–G, T, U).
  • This visualization is for demonstration only. It does not display measured survival rates per deck. See the notebook for actual data analysis and charts.
  • The prediction system uses a trained machine learning model (Random Forest, Logistic Regression, etc.) that analyzes passenger characteristics to estimate survival probability.
  • Key features include: ticket class, gender, age, fare, family size, port of embarkation, title, and deck location.
  • To use the predictor: Fill in the form on the right, click "Predict Survival", and see your results instantly.
