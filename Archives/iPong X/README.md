# iPong X - Static Site

A standalone static website showcasing the fictional "iPong X" smartphone - a parody product page with humorous specifications and comparisons.

## Features

- Fully static HTML/CSS/JS
- Bootstrap 5.3.3 for responsive design
- No build process required
- Works on GitHub Pages

## Usage

Simply open `index.html` in a web browser or deploy to any static hosting service.

### GitHub Pages Deployment

1. The site is already in the repository root
2. Enable GitHub Pages in repository settings
3. Set source to root directory
4. Access at: `https://[username].github.io/[repo]/iPong-X/`

## Files

- `index.html` - Main page
- `styles.css` - Custom styles (Bootstrap loaded from CDN)
- `assets/` - Images and icons
  - `phones/` - Phone product images
  - `icons/` - Specification icons
  - `pear-logo.png` - Brand logo

## Technologies

- HTML5
- CSS3
- Bootstrap 5.3.3 (CDN)
- Vanilla JavaScript (via Bootstrap bundle)
