# Archives

This directory contains archived projects and experiments for long-term storage.

## Contents
- **`ai/`**: Artificial Intelligence and Data Science projects (e.g., Titanic survival prediction, Crypto price forecasting).
- **`python/`**: Various standalone Python scripts, GUI applications (Tkinter), and Manim animations.
- **`web/`**: Older web projects and minigames.

*Note: To view the Jupyter notebooks in `ai/` you'll need a working Jupyter environment or a compatible viewer.*
